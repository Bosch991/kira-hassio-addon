# OpenArt Plugin

Adapter fuer OpenArt-Bildgenerierung mit Kiras bestehendem Modell, Style und World.

Konfiguration: `config/plugins/openart.yaml`

Runtime-IDs kommen aus `.env`:

- `OPENART_API_KEY`
- `OPENART_KIRA_MODEL_ID`
- `OPENART_KIRA_STYLE_ID`
- `OPENART_KIRA_WORLD_ID`
- `OPENART_DEFAULT_PROJECT_ID`
